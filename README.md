```
lumiere@wirbelwind
```

Configuration files for my ROG Zephyrus G14, aka wirbelwind.  
Please pay attention that this machine is slowly turning from a server to a desktop computer so part of its configuration is not designed to be light, but for user-friendly.  
I hate home-manager, but on this machine I tried to taste a bite.  
Using flakes for my home network environment is relatively stable and reliable (thanks to the GBA prepaid SIM card-powered hotspot).  

## Features
Full Rust utilities.  
Cleaning garbage in a routine.  
TUI agent coding tools.  
A hydra build machine at localhost:3000.  
A grafana monitor at localhost:3500.  
A sillytavern AI frontend at localhost:8000.
